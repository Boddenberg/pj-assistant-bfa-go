# Favoritos e Notificações

## Favoritos

O cliente pode salvar destinatários frequentes como favoritos para facilitar futuras transferências.

Dados de um favorito:
- **Apelido** — nome dado pelo cliente
- **Tipo de destino**: pix, ted, doc ou boleto (bill)
- **Chave PIX** (tipo e valor, se for PIX)
- **Dados bancários** (banco, agência, conta, se for TED/DOC)
- **Nome e documento do destinatário**
- **Contador de uso** — quantas vezes foi utilizado

O cliente pode criar, consultar e excluir favoritos.

## Notificações

O cliente pode consultar notificações sobre eventos da conta. Cada notificação contém:
- **Tipo** — categoria da notificação
- **Título** — resumo curto
- **Corpo** — mensagem completa
- **Canal** — meio de envio
- **Prioridade** — nível de importância
- **Lida** — se o cliente já visualizou

O cliente pode consultar suas notificações e marcar como lida.
