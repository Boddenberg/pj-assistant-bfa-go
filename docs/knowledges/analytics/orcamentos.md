# Orçamentos (Budgets)

## O que são

O cliente pode definir orçamentos mensais por categoria de gasto para controlar suas despesas.

## Dados de um orçamento

- **Categoria** — categoria de gasto (ex.: office_supplies, marketing, travel)
- **Limite mensal** — valor máximo que o cliente quer gastar nessa categoria por mês
- **Ativo** — se o orçamento está ativo ou desativado

## Funcionalidades

O cliente pode:
- Criar orçamentos para diferentes categorias
- Consultar seus orçamentos ativos
- Atualizar o valor ou desativar um orçamento
