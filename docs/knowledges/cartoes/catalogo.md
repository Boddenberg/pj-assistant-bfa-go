# Cartões de Crédito PJ — Catálogo de Produtos

## Cartões disponíveis

O Itaú oferece 4 produtos de cartão de crédito para clientes PJ:

### 1. Itaú PJ Básico
- **Bandeira:** Elo
- **Tipo:** Corporativo
- **Limite mínimo:** R$ 500
- **Limite máximo:** R$ 10.000
- **Anuidade:** Gratuita (R$ 0)
- **Benefícios:** Sem anuidade, controle de gastos
- **Descrição:** Cartão corporativo essencial sem anuidade

### 2. Itaú PJ Gold
- **Bandeira:** Visa
- **Tipo:** Corporativo
- **Limite mínimo:** R$ 5.000
- **Limite máximo:** R$ 50.000
- **Anuidade:** R$ 29,90/mês
- **Benefícios:** Seguro viagem, sala VIP nacional
- **Descrição:** Cartão empresarial com benefícios Gold

### 3. Itaú PJ Platinum
- **Bandeira:** Mastercard
- **Tipo:** Corporativo
- **Limite mínimo:** R$ 15.000
- **Limite máximo:** R$ 200.000
- **Anuidade:** R$ 59,90/mês
- **Benefícios:** Sala VIP internacional, concierge, seguro compras
- **Descrição:** Cartão premium para empresas em crescimento

### 4. Itaú PJ Virtual
- **Bandeira:** Visa
- **Tipo:** Virtual
- **Limite mínimo:** R$ 100
- **Limite máximo:** R$ 50.000
- **Anuidade:** Gratuita (R$ 0)
- **Benefícios:** Emissão instantânea, uso exclusivo online
- **Descrição:** Cartão virtual para compras online

## Pré-requisito

Para solicitar qualquer cartão, a conta precisa ter limite de crédito pré-aprovado maior que zero.
