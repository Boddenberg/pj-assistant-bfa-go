# Extrato e Transações

## Tipos de transações

O extrato bancário registra os seguintes tipos de movimentação:

| Tipo | Descrição |
|------|-----------|
| `pix_sent` | PIX enviado (débito) |
| `pix_received` | PIX recebido (crédito) |
| `debit_purchase` | Compra no débito |
| `credit_purchase` | Compra no crédito |
| `transfer_in` | Transferência recebida |
| `transfer_out` | Transferência enviada |
| `bill_payment` | Pagamento de boleto |
| `credit` | Crédito avulso |
| `debit` | Débito avulso |

## Resumo financeiro

O resumo de transações fornece:
- Total de créditos no período
- Total de débitos no período
- Saldo resultante
- Quantidade de transações
- Principais categorias de gasto

## Observação sobre PIX no crédito

Transferências PIX feitas via cartão de crédito NÃO aparecem no extrato da conta corrente. Elas são registradas exclusivamente na fatura do cartão de crédito.
